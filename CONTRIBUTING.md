# Contributing

Thanks for your interest in improving AI Traffic Attribution!

## Areas We Need Help

- **New bot patterns**: AI agents launch frequently. If you spot a new bot user-agent, submit a PR to add it to the detection patterns.
- **Alternative analytics backends**: Plausible, Umami, Matomo, PostHog — adapters for non-GA4 setups.
- **Brand lift refinements**: Better statistical methods for estimating the multiplier.
- **Dashboard improvements**: Charts, visualizations, export features.
- **Framework adapters**: Express, Fastify, Django, Rails — not just Next.js.
- **Documentation**: Guides, tutorials, case studies.

## How to Contribute

1. Fork the repo
2. Create a feature branch: `git checkout -b my-feature`
3. Make your changes
4. Submit a pull request with a clear description

## Code Style

- TypeScript for API and dashboard code
- Plain JavaScript for Cloudflare Workers (no build step required)
- Keep dependencies minimal

## Questions?

Open an issue or reach out at [getaiso.com](https://getaiso.com).
